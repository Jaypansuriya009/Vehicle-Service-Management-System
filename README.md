# Vehicle-Service-Management-System

A web-based Vehicle Service Management System developed using PHP, MySQL, HTML, CSS, JavaScript, and Apache (XAMPP).

## Features

- User Registration & Login
- Vehicle Service Requests
- Mechanic Management
- Customer Management
- Service Tracking
- Report Generation
- Admin Dashboard

## Technology Stack

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: Apache (XAMPP)

## Installation

1. Install XAMPP
2. Import vsms.sql into MySQL
3. Configure database connection in config.php
4. Start Apache and MySQL
5. Open the project in a browser
